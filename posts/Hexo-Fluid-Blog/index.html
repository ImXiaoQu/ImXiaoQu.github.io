使用Hexo搭建属于你自己的博客

欢迎来到 MineStart’s Blog，为您带来有用且免费的经验教学!
Hexo博客是我一步一步摸索过来的，期间我找了很多博主的教程，结合起来，就有了今天的教程！

前提

相信大家对于博客这个东西不陌生吧，我身边的朋友包括我同学都有属于他们自己的博客

  • MSDNicrosoft ‘s Blog
  • GuaiJie’s Blog
  • Mintimate’s Blog

等等(在我的友情链接里owo)

博客的起源:

  • 在网络上发表Blog的构想始于1998年，但到了2000年才开始真正流行；
  • 2000年博客开始进入中国，并迅速发展，但都业绩平平；
  • 2004年木子美事件，才让中国民众了解到了博客，并运用博客；
  • 2005年，国内各门户网站，如新浪、搜狐，原不看好博客业务，也加入博客阵营，开始进入博客

    From Baidu

博客推荐

市面上的博客有很多博客，这里我收集了一些常用的博客(仅针对个人意见！！)

  • WordPress
    优点：功能强大、扩展性强、对seo搜索引擎友好、适合DIY
    缺点：不能安装太多插件、备份和网站转移比较繁琐，需要涉及到数据库!
  • Typecho
    优点： 容易上手，对小白友好、简单快速的博客平台
    缺点： 主题模板少
  • Hexo
    优点：美观简洁，主题模板多
    缺点：暂时没有(doge)

那我们今天使用Hexo来搭建一个属于你自己的Blog(doge)

启动Hexo博客

hexo server 简写为hexo s

安装Hexo-Fluid主题

可按照 Hexo-Fluid 官方文档 来安装

下载 最新 release 版本 解压到 /Blog/themes 目录，并将解压出的文件夹重命名为 fluid。

然后在博客目录下创建 _config.fluid.yml，将主题的 _config.yml 内容复制进去。
